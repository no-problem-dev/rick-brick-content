## はじめに

2024年、TSMCが熊本県に建設した半導体製造工場（JASM）が本格稼働を開始しました。この動きは単なる工場誘致にとどまらず、AIチップの爆発的需要増加を背景とした**グローバル半導体サプライチェーンの構造変革**を象徴しています。

本記事では、TSMCの日本進出を起点に、半導体製造の技術的課題、AIチップ需要がサプライチェーンに与える影響、そして次世代パッケージング技術の動向を技術者の視点で解説します。

## 技術解説

### AI チップ需要と製造キャパシティ

大規模言語モデル（LLM）の訓練・推論に必要な GPU/TPU の需要は指数関数的に増加しています。NVIDIA H100 の供給不足は2023年後半から深刻化し、主要クラウドプロバイダーの調達リードタイムは6ヶ月以上に延びました。

この需要増大に対応するため、TSMCは以下の戦略を展開しています：

- **地理的分散**: 台湾集中リスクの軽減（日本・米国・ドイツに新工場）
- **先端プロセスの拡充**: 3nm/2nm ノードの量産体制強化
- **CoWoS パッケージング増産**: AI チップ向け先端パッケージングの生産能力を2倍以上に拡大

### 先端パッケージング技術

現代の AI チップ性能は、トランジスタの微細化だけでなく**パッケージング技術**に大きく依存しています。

#### CoWoS（Chip on Wafer on Substrate）

TSMCのCoWoS技術は、複数のチップレット（HBMメモリ、GPUダイ）を1つのシリコンインターポーザ上に統合します。NVIDIA H100/H200 はこの技術を採用しており、GPU とHBM3間の広帯域・低遅延通信を実現しています。

```
┌─────────────────────────────────────┐
│           パッケージ基板             │
│  ┌───────────────────────────────┐  │
│  │     シリコンインターポーザ     │  │
│  │  ┌─────┐ ┌────┐ ┌────┐      │  │
│  │  │ GPU │ │HBM3│ │HBM3│ ... │  │
│  │  │ ダイ │ │    │ │    │      │  │
│  │  └─────┘ └────┘ └────┘      │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

#### チップレットアーキテクチャ

AMDのMI300Xに代表されるチップレットアプローチは、異なるプロセスノードで製造されたダイを組み合わせることで、歩留まり向上とコスト最適化を実現します。

### JASM（熊本工場）の技術的位置づけ

JASM第1工場は **12/16nm および 22/28nm** プロセスを担当します。先端プロセスではありませんが、以下の点で重要です：

1. **車載半導体**: 自動運転AI向けのセンサー処理チップ
2. **IoTデバイス**: エッジAI推論チップの量産
3. **パワー半導体**: EV向けSiCデバイスとの組み合わせ

第2工場（2027年稼働予定）では **6/7nm** プロセスが計画されており、より高性能なAIエッジチップの製造が見込まれます。

## サプライチェーンへの影響

### レジリエンスの向上

台湾海峡リスクを考慮した地理的分散は、AIインフラの安定供給に直結します。日本政府の補助金（第1工場で約4,760億円）は、この戦略的重要性を反映しています。

### 人材とエコシステム

熊本を中心に半導体関連企業の集積が進んでおり、装置メーカー（東京エレクトロン、SCREENなど）や材料メーカー（信越化学、SUMCOなど）のサプライチェーンが強化されています。

## まとめ

半導体サプライチェーンは、AI需要の爆発的増加により歴史的な転換期を迎えています。TSMCの日本進出は、地政学的リスク分散と需要対応の両面で重要な一歩です。技術者として注目すべきは、先端パッケージング技術の進化がAIチップ性能のボトルネック解消にどこまで貢献できるかという点でしょう。
