## はじめに

Stable Diffusion、DALL-E 2、Midjourney など、現代の画像生成AIを支える中核技術が **拡散モデル（Diffusion Models）** です。2020年にHo らが発表した「Denoising Diffusion Probabilistic Models（DDPM）」は、拡散モデルを実用的な水準に引き上げた画期的な論文です。

画像生成AIの歴史は長く、VAE（変分オートエンコーダ）やGAN（敵対的生成ネットワーク）などがその先駆けとして知られています。しかしGANは学習の不安定さや、生成画像の多様性不足という課題を抱えていました。拡散モデルはこれらの問題を別のアプローチで解決し、高品質かつ多様な画像を安定的に生成する道を開きました。

この記事では、拡散モデルの数理的な仕組みを直感的に理解できるよう丁寧に解説します。

## 技術解説

### 拡散プロセスの直感的理解

拡散モデルのアイデアは非常にシンプルです。**「画像に少しずつノイズを加えて完全なランダムノイズにするプロセス（Forward Process）を学習し、その逆操作（Reverse Process）でランダムノイズから画像を生成できるようにする」**というものです。

物理現象に例えると、インクを水に垂らしたとき（Forward）、インクが徐々に拡散していく様子を思い浮かべてください。拡散モデルはこの「拡散」を時間的に逆転させる（Reverse）ことを学習します。

### Forward Process（順方向拡散）

Forward Process では、元画像 x₀ に対してT ステップにわたってガウスノイズを加えていきます。

```
q(x_t | x_{t-1}) = N(x_t; sqrt(1 - β_t) * x_{t-1}, β_t * I)
```

βₜ はノイズスケジュールと呼ばれるパラメータで、各ステップでどれだけノイズを加えるかを制御します。T ステップ後の xT は、ほぼ純粋なガウスノイズ N(0, I) になります。

重要な性質として、**任意のステップ t の xₜ を、x₀ とランダムノイズ ε から直接サンプリングできる**閉形式が存在します。

```python
# 任意ステップへのノイズ付加（簡略）
def add_noise(x0, t, noise):
    alpha_bar_t = alpha_bars[t]
    return sqrt(alpha_bar_t) * x0 + sqrt(1 - alpha_bar_t) * noise
```

### Reverse Process（逆方向拡散）とニューラルネットワーク

Reverse Process では、xT から x₀ を少しずつ復元していきます。DDPMではこの逆方向を学習するために **U-Net** アーキテクチャを使います。

ネットワークの学習目標は、各ステップで加えられたノイズ ε を予測することです（ノイズ予測）。学習済みのネットワーク εθ を使って逆方向ステップを繰り返すと、最終的に元画像に近い画像が復元されます。

```
L_simple = E_{t, x0, ε}[ ||ε - ε_θ(x_t, t)||² ]
```

この目標関数の単純さがDDPMの強みのひとつです。

### なぜ高品質な画像が生成できるのか

- **尤度最大化**：VAE と同様に変分下界（ELBO）を最大化しており、理論的な裏付けがある
- **段階的な復元**：一度に画像を生成せず、T ステップかけて徐々に洗練させる
- **柔軟な条件付け**：テキストなどの条件情報をクロスアテンションで組み込めるため、Stable Diffusion のような条件付き生成が可能

## まとめ

DDPMは、シンプルなノイズ予測という学習目標のもと、高品質な画像生成を実現した重要なマイルストーンです。その後、DDIM による高速サンプリング、LDM（Latent Diffusion Model、Stable Diffusion の基盤）での潜在空間への適用など、数多くの発展的研究が生まれました。

画像生成AI ブームの数理的な出発点として、拡散モデルへの理解を深めることは、現代生成AIを正確に把握するうえで不可欠といえます。
