## はじめに

2024年4月のAI業界を席巻したのは、**オープンソースLLMの急速な進化**でした。Meta が Llama 3 を公開し、商用利用可能な高性能モデルが誰でも使えるようになったことで、AI の民主化が新たな段階に入ったといえます。

クローズドなGPT-4 や Claude 3 が引き続き最高性能を誇る一方、オープンソースモデルはその差を急速に縮め、多くの実用的なユースケースで十分な性能を発揮するレベルに達しました。この月の動向を詳しく振り返ります。

## 主要なニュース

### Meta: Llama 3 正式公開

4月最大のトピックは **Meta Llama 3** の公開です。8B および70B パラメータの2サイズが公開され、どちらも同等サイズのオープンソースモデルを大きく上回るベンチマーク性能を記録しました。

Llama 3 の主な改善点は以下のとおりです。

- **トークナイザーの改良**: 語彙数が32K から128K に拡大し、英語以外の言語やコードの処理が大幅に向上
- **GQA（Grouped-Query Attention）**: 推論効率が改善し、より少ないGPUメモリで高速推論が可能
- **大規模データでの学習**: 15兆トークン以上のデータで学習。Llama 2 の約7倍
- **商用利用可能**: Llama 3 コミュニティライセンスにより、月間アクティブユーザー7億人未満のサービスでは無料で商用利用可能

70B モデルは GPT-3.5 Turbo を多くのベンチマークで上回っており、APIコストを抑えたい企業にとって魅力的な選択肢となっています。

### Mixtral と Mistral AI のエコシステム

フランスのスタートアップ Mistral AI もオープンソース勢として存在感を高めています。

- **Mixtral 8x22B**: MoE（Mixture of Experts）アーキテクチャを採用した大規模モデルを公開。64B パラメータ相当の性能を持ちながら、推論時は一部のエキスパートのみ使用するため効率的
- **Mistral Large**: クローズドAPIとして提供。GPT-4 に近い性能をより低コストで提供
- **La Plateforme**: Mistral のAPI プラットフォームが欧州のAI規制対応を強調し、欧州企業からの引き合いが増加

### HuggingFace エコシステムの発展

**HuggingFace** はオープンソースAIの中心地として、プラットフォームの機能拡張を続けました。

- **Inference Endpoints**: セルフホストのモデルをワンクリックでAPIとして公開できる機能が強化
- **PEFT と TRL の更新**: LoRA、QLoRA などのファインチューニングツールが使いやすくなり、カスタムモデル作成の敷居がさらに低下
- **Open LLM Leaderboard**: ベンチマーク評価が充実し、モデル選定の参考情報が増加

### ローカルLLM実行ツールの普及

オープンソースモデルの高性能化に伴い、PC 上でLLMを動かすツールも普及が加速しました。

- **Ollama**: macOS / Linux / Windows に対応し、`ollama run llama3` などシンプルなコマンドでモデルを実行可能
- **LM Studio**: GUIベースでモデルをダウンロードし、チャットUIやAPIサーバーとして使えるアプリ
- **Jan**: オープンソースの ChatGPT 代替として機能するデスクトップアプリ
